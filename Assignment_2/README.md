# Assignment 2 - UCampus Cafeteria Management System

## Description
This assignment implements the **UCampus Cafeteria Management System** using Java. It demonstrates the use of classes, objects, instance variables, static variables, instance methods, and static methods.

## Features
- Add food items
- Calculate final price including service charge
- Display food item details
- Change cafeteria service charge
- Display cafeteria details
- Track total number of food items

## Classes
- FoodItem
- Main

## Concepts Used
- Classes and Objects
- Instance Variables
- Static Variables
- Instance Methods
- Static Methods
- Constructors
- User Input (Scanner)
